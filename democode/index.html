Fast Dot Product – Benchmark Demo Page

Simple example code on how to run dot product calculations on the web platform, but fast. Accelerated by GPU, CPU vector instructions (WebAssembly, close to native), and JIT optimizer instruction/memory alignment.

Implementations, ordered by performance:

  1. WebAssembly, using SIMD instruction set (~380% faster*)
  2. JIT-optimized JavaScript (~300% faster*)
  3. Baseline: Standard JavaScript
*compared to baseline
